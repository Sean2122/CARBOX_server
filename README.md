
# CARBOX Application

This is a .NET 8.0 web application for managing car sharing services.

## Features
- Car management
- Route planning
- Station management
- Ride ordering system
- MQTT integration
- MongoDB database

## Technologies Used
- ASP.NET Core 8.0
- MongoDB Driver
- MQTTnet
- Swashbuckle (Swagger)
- TimeZoneConverter

## Getting Started
Run the application using:
```
dotnet run
```
